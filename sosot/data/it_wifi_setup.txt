# 사내 Wi-Fi 연결 가이드

## 대상
전 직원 (신규 입사자 포함)

## 연결 방법

### Windows
1. 작업표시줄 우측 하단의 Wi-Fi 아이콘을 클릭합니다.
2. 네트워크 목록에서 "CORP-WIFI"를 선택합니다.
3. 사번과 AD 비밀번호를 입력합니다.
4. 인증서 경고가 나타나면 "연결"을 클릭합니다.

### Mac
1. 메뉴바의 Wi-Fi 아이콘을 클릭합니다.
2. "CORP-WIFI"를 선택합니다.
3. 사번과 AD 비밀번호를 입력합니다.
4. 인증서 신뢰 팝업에서 "계속"을 클릭합니다.

## 게스트 네트워크
방문객은 "GUEST-WIFI"에 접속하며, 안내 데스크에서 일회용 비밀번호를 발급받으세요.

## 문제 해결
- 연결이 안 될 경우: AD 비밀번호가 만료되었는지 확인하세요.
- 속도가 느린 경우: 5GHz 대역("CORP-WIFI-5G")으로 전환해 보세요.
- 그래도 해결되지 않으면 IT 헬프데스크(내선 1234)로 연락해 주세요.

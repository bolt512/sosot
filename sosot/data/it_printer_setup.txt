# 사내 프린터 설치 가이드

## 대상
프린터를 새로 설치해야 하는 직원

## 공용 프린터 연결

### Windows
1. 설정 > 장치 > 프린터 및 스캐너를 엽니다.
2. "프린터 또는 스캐너 추가"를 클릭합니다.
3. 목록에서 해당 층의 프린터를 선택합니다.
   - 3층: PRINTER-3F-COLOR (컬러)
   - 4층: PRINTER-4F-MONO (흑백)
   - 5층: PRINTER-5F-COLOR (컬러)
4. 드라이버가 자동 설치됩니다.

### Mac
1. 시스템 설정 > 프린터 및 스캐너를 엽니다.
2. "+" 버튼을 클릭합니다.
3. 해당 층의 프린터를 선택하고 "추가"를 클릭합니다.

## 인쇄 할당량
- 월 흑백 500매, 컬러 100매까지 무료입니다.
- 초과 시 팀 예산에서 차감되며, 팀장에게 알림이 전송됩니다.

## 문제 해결
- 프린터가 목록에 나타나지 않을 경우: 같은 네트워크(CORP-WIFI)에 연결되어 있는지 확인하세요.
- 인쇄가 안 될 경우: 프린터 전원 상태와 용지함을 확인하세요.
- 드라이버 설치 오류: IT 헬프데스크(내선 1234)에 요청하세요.

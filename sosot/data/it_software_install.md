# 소프트웨어 설치 가이드

## 대상
업무용 소프트웨어 설치가 필요한 직원

## 사내 소프트웨어 센터
사내에서 승인된 소프트웨어는 **소프트웨어 센터**를 통해 설치합니다.

1. 바탕화면의 "소프트웨어 센터" 아이콘을 실행합니다.
2. 필요한 프로그램을 검색합니다.
3. "설치" 버튼을 클릭하면 자동으로 설치됩니다.

## 사용 가능한 소프트웨어 목록
| 분류 | 프로그램 |
|------|----------|
| 오피스 | Microsoft Office 365, 한컴오피스 |
| 메신저 | Mattermost, Zoom |
| 개발도구 | VS Code, Git, Python |
| 보안 | 백신(V3), DLP 클라이언트 |
| 유틸리티 | 7-Zip, Adobe Reader |

## 목록에 없는 소프트웨어
1. IT 헬프데스크에 **소프트웨어 설치 요청서**를 제출합니다.
2. 보안 검토 후 승인되면 설치가 진행됩니다 (영업일 기준 2~3일 소요).
3. 라이선스가 필요한 경우 팀 예산에서 처리됩니다.

## 주의사항
- 임의로 소프트웨어를 설치하면 보안 정책 위반입니다.
- P2P 프로그램, 게임 등 업무 외 소프트웨어는 설치가 금지됩니다.
- 문의: IT 헬프데스크(내선 1234)

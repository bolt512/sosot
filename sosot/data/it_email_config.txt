# 사내 이메일 설정 가이드

## 대상
신규 입사자, 이메일 클라이언트 재설정이 필요한 직원

## 웹메일 접속
- 주소: https://mail.company.co.kr
- 사번과 AD 비밀번호로 로그인

## Outlook 설정
1. Outlook을 실행하고 "계정 추가"를 선택합니다.
2. 이메일 주소를 입력합니다: 사번@company.co.kr
3. 비밀번호를 입력하면 자동 구성이 진행됩니다.
4. 자동 구성이 실패할 경우 수동 설정:
   - 수신 서버(IMAP): imap.company.co.kr, 포트 993, SSL
   - 발신 서버(SMTP): smtp.company.co.kr, 포트 587, TLS
   - 인증: AD 계정 사용

## 모바일 이메일 설정
1. 휴대폰 설정 > 계정 > 계정 추가 > Exchange를 선택합니다.
2. 이메일: 사번@company.co.kr
3. 서버: mail.company.co.kr
4. 도메인: CORP
5. 사번과 AD 비밀번호를 입력합니다.

## 주의사항
- 메일함 용량은 5GB이며, 초과 시 수신이 차단됩니다.
- 대용량 파일은 사내 파일 공유 시스템을 이용하세요 (첨부파일 최대 25MB).
- 외부 메일 자동 전달은 보안 정책상 금지되어 있습니다.
